## 🚀 Co to jest EduGuard?

EduGuard to aplikacja dla Windows, która:
- 🔒 Blokuje gry i programy rozrywkowe
- 🧮 Wymaga rozwiązania zadań matematycznych (kl. 1–8)
- ⏱️ Odblokowuje grę proporcjonalnie do czasu nauki
- 👨‍👩‍👧 Obsługuje wiele profili dzieci
- 📊 Wysyła tygodniowy raport e-mail do rodzica

## 🛡️ Bezpieczeństwo

4-warstwowa ochrona — dziecko nie obejdzie blokady przez 
Task Manager, CMD ani PowerShell.

## 💻 Wymagania

- Windows 10 / Windows 11
- Brak wymagań internetowych (działa offline)
